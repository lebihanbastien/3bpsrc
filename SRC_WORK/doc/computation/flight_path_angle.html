DESCRIPTION 

-------------------------------------------------------------------------%
 flight_path_angle(yvc, center) 

 Flight path angle in radian. If the state in position/velocity is (x, v),
 the sinus of the flight path angle with respect to the center x0 
 is given by:

   sin(fpa)  = - (x - xO).v / (|x - x0|*|v|)
